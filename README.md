# train-ticket-dispenser

## Problem statement

Design an automated system for purchasing train tickets for domestic travel. The buyer enters the distance to the destination (in tens of kilometers). The ticket cost is displayed on 7-segment displays. The currency used is EURO. The maximum price for a ticket is 50 EURO. The machine accepts the required amount in banknotes or coins and issues the ticket and, if necessary, the change.

Situations such as lack of tickets, insertion of an amount smaller than the ticket cost, or inability to return change are indicated by light signals.
